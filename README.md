# SEO Content Prompt System  
### Local SEO + AI + Prompt Engineering Project

---

## Project Overview

This project demonstrates a **structured SEO content creation system** built using **AI and prompt engineering**, following real-world **SEO agency workflows**.

Instead of writing random blog posts, this system uses:
- SEO planning
- Content clusters (pillar + supporting blogs)
- Search intent targeting
- Local SEO optimization

The goal is to show how AI can be guided using **well-designed prompts** to generate **ranking-focused, business-ready content**.

---

## Business Context

- **Business Type:** Dental Clinic  
- **Location:** Ayodhya  
- **Target Audience:** Local patients searching for dental services on Google  
- **Primary Goal:** Increase organic traffic and appointment inquiries through SEO blogs  

All content is written as if it will be published on a **real dental clinic website in Ayodhya**.

---

## SEO Content Strategy

This project follows a **content cluster strategy**, which is commonly used by SEO agencies and SaaS companies.

### 1. Pillar Blog
A long-form, authoritative article that targets the main keyword:

**Example Pillar Topic:**  
> *Best Dental Clinic in Ayodhya – Services, Cost & What to Expect*

Purpose:
- Build topical authority
- Target high-value local keywords
- Act as the main internal linking hub

---

### 2. Supporting Blogs (Content Cluster)

Shorter blogs that support the pillar article and target specific search intents such as:
- Cost-related searches
- Service-specific queries
- Informational questions

Examples:
- Dental Implant Cost in Ayodhya  
- Root Canal Treatment in Ayodhya – Procedure & Cost  
- How to Choose the Right Dentist in Ayodhya  

Each supporting blog internally links back to the pillar blog.

---

## Prompt Engineering System

The core of this project is the **prompt system**, which is designed to be **reusable and scalable**.

### Prompt Categories

- **Pillar Outline Prompt**  
  Generates SEO-optimized H1–H3 blog structures.

- **Pillar Content Prompt**  
  Converts outlines into long-form, human-readable, SEO-friendly content.

- **Supporting Blog Prompt**  
  A reusable template that generates multiple supporting blogs by changing the topic and intent.

This separation reflects how professional teams control AI output quality and consistency.

## SEO & Search Intent Logic

Each piece of content is created with a clear **search intent**:

- **Informational:**  
  Users seeking knowledge (e.g., teeth whitening safety)

- **Commercial:**  
  Users comparing options (e.g., best dental clinic in Ayodhya)

- **Transactional:**  
  Users ready to take action (e.g., dental implant cost in Ayodhya)

Local SEO is achieved by naturally combining:
- Service keywords
- City name (Ayodhya)
- Patient-focused language
